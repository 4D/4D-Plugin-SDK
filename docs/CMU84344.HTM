PA_GetCarbonPrintSettings

version 2003

PA_GetCarbonPrintSettings → void*

Parameter Type Description
This command does not require any parameters
Function result void* ← current printing settings

Description

The PA_GetCarbonPrintSettings command returns the current printing settings used by 4D. The returned value is a pointer to the carbon PMPrintSettings object.
