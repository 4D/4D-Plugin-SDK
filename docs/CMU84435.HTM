PA_Dial4DSetArrayTextFromResource

version 2003

PA_Dial4DSetArrayTextFromResource (dialog; arrayName)

Parameter Type Description
dialog PA_Dial4D → Dialog reference
arrayName char* → Name of the text array to build

Description

The routine PA_Dial4DSetArrayTextFromResource builds a text array from a Macintosh resource (STR# or MENU), then assigns it to arrayName . This is a quick way to build a text array.

See Also

No reference.

Error Handling

None.
