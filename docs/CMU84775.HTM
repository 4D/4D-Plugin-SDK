PA_QueryTime

version 2003

PA_QueryTime (queryRef; tNum; fNum; qOp; qComp; value)

Parameter Type Description
queryRef PA_QueryRef → Reference of the Query
tNum short → Table number
fNum short → Field number to search
qOp PA_QueryOperator → The query operator
qComp PA_QueryComparison → The query comparison
value long → Value to search in field fNum

Description

The routine PA_QueryTime adds one line of search to the current query being built after a call to PA_OpenQuery . It adds a line of search in a field of type time (which is a long field).

If this is the first line of a search (the first one after calling PA_OpenQuery ), qOp must be set to eQO_NoOperator .

Refer to Building a query with the API to learn how to use the routine.
