PA_Dial4DHighlightText

version 2003

PA_Dial4DHighlightText (dialog; variable; startSelection; endSelection)

Parameter Type Description
dialog PA_Dial4D → Dialog reference
variable char* → Text variable name to access
startSelection short → Start of the selection in the text
endSelection short → End of the selection in the text

Description

The command PA_Dial4DHighlightText hilights the specified text selection in the specified variable.
