PA_Dial4DGotoPage

version 2003

PA_Dial4DGotoPage (dialog; pageNum)

Parameter Type Description
dialog PA_Dial4D → Dialog reference
pageNum short → Page number

Description

The routine PA_Dial4DGotoPage forces the dialog to go to a page. This is the equivalent of the 4th Dimension command GOTO PAGE ( pageNum ).

See Also

GOTO PAGE .

Error Handling

None.
