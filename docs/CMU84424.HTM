PA_Dial4DGetCurrentPage

version 2003

PA_Dial4DGetCurrentPage (dialog) → short

Parameter Type Description
dialog PA_Dial4D → Dialog reference
Function result short ← Current page number

Description

The routine PA_Dial4DGetCurrentPage retrieves the number of the page that the dialog is currently on. You can use this command, for example, to update the value of the variables that are contained only on this page for optimization reasons.
