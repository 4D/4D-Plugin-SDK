PA_Dial4DSetWindowSize

version 2003

PA_Dial4DSetWindowSize (dialog; width; height)

Parameter Type Description
dialog PA_Dial4D → Dialog reference
width long → New vertical size of the dialog
height long → New horizontal size of the dialog

Description

The routine PA_Dial4DSetWindowSize allows to change the dialog size.

See Also

No reference.

Error Handling

None.
