PA_Dial4DDial4DFromWindow

version 2003

PA_Dial4DDial4DFromWindow (window) → PA_Dial4D

Parameter Type Description
window PA_WindowRef → Window reference
Function result PA_Dial4D ← Dialog context reference

Description

The routine PA_Dial4DDial4DFromWindow returns the dialog context reference from a given window reference. It returns 0 if the specified window is not handled by a 4D dialog.
