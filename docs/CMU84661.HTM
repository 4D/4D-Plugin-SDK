PA_Open4DPreferencesMacResFile

version 2003

PA_Open4DPreferencesMacResFile → short

Parameter Type Description
This command does not require any parameters
Function result short ← The resource file reference of the preference file

Description

The routine PA_Open4DPreferencesMacResFile returns the resource file reference number of the 4th Dimension preferences file.

With that number you can access to the resources of the 4D preferences file.

IMPORTANT NOTE

Each call to PA_Open4DPreferencesMacResFile must be balanced with a call to PA_ClosePreferences .

NOTE

4D preferences is a Macintosh Resource File; you must use Altura's Mac2Win to access resources under Windows.

WARNING

Modifying 4D preferences resources or storing your preferences in 4D preferences file is not recommended. A plug-in should create and use its own preferences file.

Example

Read a 'xPrf' resource in 4D resource file.

Compile for MacOS or Windows using Mac2Win. 

   short   previous;
   Handle   hpref;
 
   previous = CurResFile();
   UseResFile( PA_Open4DPreferencesMacResFile() );
   hpref = GetResource( 'xPrf', 15000 );
   PA_ClosePreferences();
   UseResFile(previous);
